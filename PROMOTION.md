```
# AI Workbench - AI-powered Obsidian Plugin 🤖

Hi Obsidian community! 👋

I'm excited to share my new plugin: **AI Workbench** - an AI-powered workbench for Obsidian that makes note processing effortless!

## ✨ What can it do?

- **Quick Actions**: Summarize, outline, translate, format, generate mind maps
- **Custom Prompts**: Create your own AI processing actions
- **25+ Preset Templates**: Xiaohongshu, short videos, WeChat articles, and more
- **Selected Text Processing**: Process only what you need
- **Backup Management**: Never lose your work
- **Usage Statistics**: Track token usage and costs

## 🎯 Key Features

1. **One-Click Processing**: Press `Ctrl+Alt+S` to instantly summarize your note
2. **Customizable**: Create your own prompts for specific workflows
3. **Safe & Private**: API keys stored locally, no data collection
4. **Open Source**: Fully transparent and auditable code

## 📦 Installation

Download from [GitHub Releases](https://github.com/zfz584521-collab/obsidian-ai-workb/releases/tag/v0.1.0)

1. Download `main.js`, `manifest.json`, and `styles.css`
2. Create `.obsidian/plugins/ai-workbench/` in your vault
3. Copy the files there
4. Restart Obsidian and enable the plugin

## ⚙️ Configuration

Supports OpenAI-compatible APIs:
- OpenAI API
- Claude API (via compatibility layer)
- Local Ollama
- Various proxy services
```

## 知乎发布

**标题：**
```
Obsidian AI 插件推荐：AI Workbench，让笔记处理效率提升10倍
```

**内容：**
```
作为一名重度 Obsidian 用户，我一直希望能有一个高效的 AI 插件来处理笔记。经过一番摸索，我开发了一个名为「AI Workbench」的插件，今天分享给大家。

## 一、为什么需要 AI Workbench？

在知识管理过程中，我们经常需要：
- 总结长篇文章
- 提取笔记要点
- 翻译外语内容
- 优化文字表达
- 生成思维导图

手动处理这些任务非常耗时。AI Workbench 的目标就是通过 AI 自动化这些重复性工作。

## 二、核心功能介绍

### 1. 快捷操作

支持多种一键操作：
- **总结**：快速提炼核心观点
- **大纲**：自动生成文章结构
- **翻译**：中英互译
- **格式化**：优化排版和表达
- **思维导图**：可视化内容结构

### 2. 自定义 Prompt

可以根据自己的需求创建专属处理模板。比如：
- 学术论文摘要
- 商业计划书框架
- 个人日记润色

### 3. 预设模板库

内置25+专业模板，覆盖：
- 内容创作（小红书、短视频、公众号）
- 学习笔记
- 工作文档
- 等等

## 三、实际应用场景

### 场景1：读书笔记

以往看完一本书，需要手动整理要点。现在：
1. 复制书籍内容到 Obsidian
2. 按 `Ctrl+Alt+S` 一键总结
3. 生成结构化的读书笔记

### 场景2：会议记录

会议结束后：
1. 将会议记录粘贴到笔记
2. 使用「提取重点」功能
3. 生成清晰的会议摘要

### 场景3：内容创作

写小红书笔记时：
1. 写好初稿
2. 使用「小红书润色」模板
3. 自动优化表达，添加表情

## 四、技术特点

- **本地优先**：API 密钥存储在本地，不上传任何数据
- **开源透明**：代码完全开源，可审计
- **兼容性强**：支持 OpenAI、Claude 等多种 API
- **性能优化**：快速响应，流畅体验

## 五、安装使用

### 安装步骤

1. 访问 GitHub Releases 页面
2. 下载 `main.js`、`manifest.json`、`styles.css`
3. 放到 `.obsidian/plugins/ai-workbench/` 目录
4. 重启 Obsidian 并启用插件

### 配置 API

在设置中填写：
- API 端点
- API 密钥
- 模型名称（推荐 gpt-4o-mini）

## 六、未来规划

- 支持更多 AI 模型
- 流式输出
- 模板市场
- 团队协作功能

## 总结

AI Workbench 让 Obsidian 的 AI 能力真正落地，提升了我的工作效率。如果你也是 Obsidian 用户，强烈推荐试试！
```
